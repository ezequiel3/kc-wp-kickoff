****HTML ICON SET****
Compiled using GIMP 2.8

**Multilayered favicon**

The Multilayered favicon is a precompiled icon file that contains the main image in different resolutions.
The browser just displays the one that it needs.

Content:
-favicon64.png
-favicon32.png
-favicon24.png
-favicon16.png

HTML Code:
<link href="images/meta/favicon.ico" type="image/vnd.microsoft.icon" rel="icon" sizes="16x16 32x32 48x48 64x64" />


**Apple Touch icon**
Icon used for Iphone/Ipad shorcuts

HTML Code:
<link href="images/meta/apple-touch-icon.png" rel="apple-touch-icon" />

**IMG SRC**
Default source image for standard metadata purposes.
This icon can also be used for facebook previews.

HTML Code:
<link href="images/meta/img-src.png" rel="image_src" />
